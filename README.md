## Usage example

Clone the project

```bash
  https://github.com/akecha/ft_printf.git
```

Go to the project directory

```bash
  cd ft_printf
```

Install dependencies

```bash
  brew install clang
  brew install make
```

Compile

```bash
  make
```
Demo

```bash
  gcc -Wall -Werror -Wextra your_main.c libftprintf.a
```
